# blogapp

一个flutter开发的博客APP，管理个人网的WordPress项目。

### 目前实现的功能

#### 1.博客文章列表查看，包括下拉刷新，上拉加载。

#### 2.博客文章详情页，展示支持HTML格式

#### 3.新建博客
